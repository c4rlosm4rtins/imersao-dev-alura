O que é um Sistema Operacional?

Um sistema operacional (SO) é um software que gerencia o hardware do computador. Recursos de hardware como entrada e saída e alocação de memória são gerenciados pelo SO.

Ele é a interface entre o usuário e o hardware do computador.
